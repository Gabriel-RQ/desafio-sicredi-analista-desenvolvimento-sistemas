# Desafio Técnico: Analista de Desenvolvimento de Sistemas

Repositório contendo as soluções para os desafios propostos para a vaga de Analista de Desenvolvimento de Sistemas do Sicredi Conexão.

## Primeiros Passos

O primeiro passo para execução do código dos desafios é clonar o repositório localmente. Basta rodar o comando abaixo, a partir da pasta desejada:

```bash
git clone https://github.com/Gabriel-RQ/desafio-sicredi-analista-desenvolvimento-sistemas
```

Feito isso, o código dos desafios estará disponível localmente. Seguir instruções nas pastas do [desafio 1](./desafio-1/) e [desafio 2](./desafio-2/) para execução do código.

## Desafio 1

Automação desenvolvida em Python para coletar informações do *website* do [Sicredi Conexão](https://www.sicrediconexao.com.br).

A solução do desafio e instruções para execução podem ser encontradas em [desafio-1/](./desafio-1/).

Recomenda-se abrir a pasta do desafio em um editor de texto ou IDE, ou defini-la como pasta atual com `cd ./desafio-1`.

## Desafio 2

API RESTful desenvolvida com PHP (Laravel), implementando autenticação por JWT e um CRUD de Associados.

A solução do desafio e instruções para execução podem ser encontradas em [desafio-2/](./desafio-2/).

Recomenda-se abrir a pasta do desafio em um editor de texto ou IDE, ou defini-la como pasta atual com `cd ./desafio-2`.
